# Election Related Analysis System (ERAS)

## Description
A web-based Election Related Analysis System developed using Python Flask, MySQL, HTML, CSS, JavaScript and ReactJS.

## Features
- User Authentication
- Forgot Password
- Settings Management
- Dashboard
- Party Analysis
- Election Data Management
- Secure Database Integration

## Technologies Used
- Python Flask
- MySQL
- HTML
- CSS
- JavaScript
- ReactJS
